# Origin.js 是什么

Origin.js 是一个简单而强大的 Web 应用框架，提供一套基于 Vue.js 的完整 Web 项目解决方案，让开发者更加专注于业务逻辑的开发。

# 特性

Origin.js 集成实用的插件和开发工具，支持如下特性，使得编码和开发更加容易：

  • Vue3 (opens new window) & Vite2 (opens new window)
  • Vue i18n (opens new window) & Vuex (opens new window)
  • 自动导入使用 css、sass、less 或 stylus 定义的全局样式
  • 自动引入 Vue 组件
  • 基于项目文件的路由配置和布局渲染
  • 支持读取 yaml/xml/xlsx/... 文件作为 ES 模块
  • Markdown 支持
  • Webpack 到 Vite 的转换工具 (opens new window)
  • Vue2 升 Vue3 的迁移工具 (opens new window)

# 示例

你可以通过 Vue3 project demo (opens new window) 试用 Origin.js 提供的插件，参考项目模板 Origin.js templates (opens new window) 快速了解使用 Origin.js CLI 创建的初始应用的结构。
